# myPOS (draft released 0.9)
myPOS Corpus (Myanmar POS Corpus) for Myanmar language NLP Research and Developments

## Introduction
The myPOS Corpus (Myanmar POS Corpus) is a 10,998 sentences (242,303 words) manually word segmented and POS tagged corpus developed for Myanmar language NLP research and developments. We collected Myanmar sentences from Wikipedia that include various area such as economics, history, news, politics and philosophy.
